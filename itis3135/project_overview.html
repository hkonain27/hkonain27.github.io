Client Project Overview

Project Overview:

  • The application will be a website for Girls Who Code (GWC) at UNC Charlotte. The primary purpose of this website is to provide a platform for empowering female students in computer science by developing technical and professional skills while fostering a supportive community.
  • The website will serve as a hub for members to access information about upcoming events, meetings, and opportunities. It will also be an informative resource for anyone interested in joining the organization or learning more about its activities and mission.
  • The site will provide an introduction to Girls Who Code at UNC Charlotte, the goals of the organization, member benefits, and upcoming events such as workshops, panels, and networking opportunities.

Page Design:

  • Index page: The index page will introduce the Girls Who Code organization and its mission at UNC Charlotte.
  • E-Board page: This page will showcase the members of the 2024-25 executive board and their roles.
  • Events page: This page will highlight upcoming events, workshops, and technical discussions hosted by GWC.
  • Membership & Join Us: This page will explain how students can become members and the benefits of joining the organization.
  • Partner With Us: Information on how businesses can partner with GWC for events, workshops, and other initiatives.

Dynamic Functionality on the Website:

The dynamic functionality that I plan on implementing using JavaScript are:

1. Event RSVP forms for the "Events" page

2. Interactive member signup forms for the "Join Us" page

3. Member calendar with dynamic event updates on the "Calendar" page
